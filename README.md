# aspireLiens.sh

- Description: 
   - Script qui aspire les liens d'une page web et les stock dans un fichier texte/html

- Utilsiation:
   - aspireLiens.sh url

- Exemples:
   - aspireLiens.sh http://wwww.google.fr/q=dino213dz

- Utilsiation:
